## 3.3 索引的操作

### 3.3.1 创建一个索引

```json
# 创建一个索引
PUT /person
{
    "settings"{
    	"number_of_shards": 5,
    	"number_of_replicas": 1
	}
}
```

### 3.3.2 查看索引信息

```json
# 查看索引信息
GET /person
```

> 注意: 这里面要让你的鼠标光标放在GET这一行上面才行,否则执行空了
